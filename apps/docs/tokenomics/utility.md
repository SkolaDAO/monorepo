# Token Utility (Planned)

::: warning Future Feature
The $LEARN token is not yet live. This page describes the planned design for when we have sufficient platform usage and liquidity to launch responsibly.
:::

## Why a Token?

The token will create aligned incentives:

1. **Creators stake to publish** - Token demand from creators
2. **Learners get discount** - Token demand from learners  
3. **Stakers earn fees** - Long-term holding incentive
4. **Buyback & burn** - Supply reduction over time

No inflationary rewards. Real utility, real demand.
